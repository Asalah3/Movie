# Movie
Movies Application

Mobile Native Movies application is an application for Displaying Box Office Movies.

## Technologies Used:
* Swift
* MVVM Architecture Pattern
* Fetching data from API
* GCD
* CoreData
* CocoaPods (SDWebImage)
* UIKit & Constraints.
